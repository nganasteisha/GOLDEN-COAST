Танцювальна зона Golden Coast

Танцювальна зона Golden Coast створена для гостей, які люблять активний відпочинок, сучасну музику та вечірні розваги. Тут можна приємно провести час, насолодитися музикою та відчути атмосферу святкового відпочинку.

На території розташовані сцена для виступів, комфортні місця для відпочинку, бари з напоями та окремі зони для гостей комплексу. У вечірній час територія освітлюється сучасним освітленням та створює особливий настрій для відпочинку.

Для гостей доступні тематичні заходи, виступи DJ, танцювальні програми та окремі зони для комфортного проведення часу.

Головна танцювальна зона

Основна територія для проведення вечірок, музичних програм та активного відпочинку.

Зона обладнана сучасним звуковим обладнанням, сценою та комфортними місцями для гостей.

Для гостей доступні напої, місця для відпочинку та додатковий сервіс.

Місткість: до 250 гостей

Вартість входу: від 700 грн

VIP зона

Окрема зона для гостей, які бажають більш комфортного та приватного відпочинку.

Для гостей доступні зручні дивани, окреме обслуговування та комфортні місця для відпочинку.

Місткість: до 40 гостей

Вартість: від 1200 грн

Вечірні DJ заходи

На території комплексу регулярно проводяться музичні програми та тематичні заходи.

Для гостей доступні музика, напої та спеціальні зони для відпочинку.

Вхід: 18+

Вартість: від 1000 грн
