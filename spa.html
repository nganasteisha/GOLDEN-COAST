SPA та оздоровча зона

SPA зона Golden Coast — це місце для повного відпочинку, відновлення сил та приємного проведення часу. Тут створена комфортна атмосфера для гостей, які бажають розслабитися та насолодитися спокійним відпочинком.

Інтер’єр SPA комплексу оформлений у сучасному стилі з використанням теплого освітлення, натуральних матеріалів та комфортних зон для відпочинку. Кожна деталь створює затишну атмосферу для гостей комплексу.

Для гостей доступні фінська сауна, масажні процедури, джакузі, ароматерапія та спеціальні зони відпочинку після процедур.

Фінська сауна

Фінська сауна є однією з найпопулярніших зон комплексу для відпочинку та відновлення сил.

Інтер’єр оформлений у сучасному стилі з використанням натурального дерева та теплого освітлення.

Висока температура допомагає розслабитися, зняти втому та покращити самопочуття.

Для гостей доступні ароматичні олії, рушники, душова зона та комфортне місце для відпочинку.

Місткість: до 10 гостей

Вартість: від 700 грн / година

Масажна зона

Професійні процедури допомагають відновити організм та забезпечують повне розслаблення.

Усі процедури проводяться спеціалістами з використанням якісних засобів та ароматичних олій.

Для гостей доступний класичний масаж, антистрес процедури та ароматерапія.

Після процедур гостям доступні напої та комфортна зона відпочинку.

Тривалість: 60 хвилин

Вартість: від 1200 грн

Зона джакузі

Комфортна зона відпочинку з теплою водою та приємною атмосферою для гостей комплексу.

Джакузі розташоване у сучасній зоні відпочинку зі стильним оформленням.

Для гостей доступні рушники, комфортні місця для відпочинку та додатковий сервіс.

Ця зона чудово підходить для відпочинку після процедур.

Місткість: до 8 гостей

Вартість: від 900 грн / година

Кімната відпочинку

Кімната відпочинку створена для гостей після SPA процедур та оздоровчих програм.

Інтер’єр оформлений для створення максимально комфортної атмосфери.

Гостям доступні чай, ароматерапія та комфортні місця для відпочинку.

Це чудове місце для спокійного проведення часу після процедур.

Доступ: входить у пакет послуг

Вартість пакета: від 2200 грн
