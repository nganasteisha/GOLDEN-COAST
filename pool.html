Басейн Golden Coast

Басейн Golden Coast — це сучасна зона відпочинку для гостей комплексу, де можна комфортно провести час біля води та насолодитися приємною атмосферою. Тут поєднуються затишок, сучасний дизайн та комфортні умови для відпочинку.

Для гостей доступні лежаки, зони відпочинку, бар із напоями та окремі місця для спокійного проведення часу. У вечірній час територія має красиве освітлення, яке створює особливу атмосферу відпочинку.

Головний басейн

Головний басейн комплексу є центральною зоною для денного відпочинку та проведення часу біля води.

Басейн обладнаний сучасною системою очищення води та комфортною температурою.

Для гостей доступні лежаки, бар, рушники та окремі зони відпочинку.

Місткість: до 120 гостей

Доступ входить у проживання

Вартість: від 900 грн / день

VIP зона біля басейну

Окрема зона для гостей, які бажають більш приватного та комфортного відпочинку.

Для гостей доступні м’які дивани, комфортні місця для відпочинку та додатковий сервіс.

Місткість: до 30 гостей

Вартість: від 2500 грн / день

Вечірні заходи біля басейну

У вечірній час на території комплексу проводяться тематичні заходи та розважальні програми.

Для гостей доступна музика, напої та окремі зони для відпочинку.

Вхід: 18+

Вартість: від 1200 грн
