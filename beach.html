Пляжна зона Golden Coast

Пляжна зона Golden Coast — це комфортне місце для відпочинку на узбережжі моря. Для гостей створені всі умови для приємного проведення часу та спокійного відпочинку біля води.

Територія пляжу обладнана сучасними лежаками, зручними місцями для відпочинку та окремими зонами для гостей комплексу. У вечірній час територія має красиве освітлення та затишну атмосферу.

Для гостей доступні напої, рушники, пляжний сервіс та додаткові зони для відпочинку.

Головна пляжна зона

Головна пляжна зона комплексу створена для комфортного відпочинку біля моря.

Територія обладнана лежаками, пляжними навісами та місцями для відпочинку.

Для гостей доступні напої, рушники та обслуговування території.

Місткість: до 150 гостей

Доступ входить у проживання

Вартість: від 700 грн / день

VIP пляжна зона

Окрема зона для гостей, які бажають більш комфортного та приватного відпочинку.

Для гостей доступні м’які дивани, зручні лежаки та окремі місця для відпочинку.

Місткість: до 35 гостей

Вартість: від 1600 грн / день

Вечірні заходи на пляжі

У вечірній час на території комплексу проводяться тематичні заходи та розважальні програми.

Для гостей доступні музика, напої та окремі місця для відпочинку.

Вхід: 18+

Вартість: від 950 грн
